## 摘要 / Summary

將專案從「GensparkLike Skill Exchange Platform（技能交換平台）」轉向為「PM Evolutionary Ecosystem（AI 產品經理工作平台）」，聚焦 PM / PM 的 AI 文件自動化需求。
Pivot from "GensparkLike Skill Exchange Platform" to "PM Evolutionary Ecosystem (AI Product Manager Work Platform)", focusing on AI-powered document automation for product managers.

## 範圍 / Scope

### 包含 / Included

1. `index.html` — 完全重寫為 PM EVO 首頁（CSS 內聯化，移除外部 style.css / main.js）
2. `status/index.html` — 網站地圖同步更新（assets/ 目錄移除）
3. `workspace/index.html` — 標題改為「PM EVO AI 工作空間 0.1」
4. `openspec/` 文件體系建立（proposal / tasks / specs / design）

### 不包含 / Excluded

1. 實際後端功能（認證、模板庫、AI 工作流）
2. 會議錄音整理的真實 API 串接
3. workspace 頁面的完整 PM EVO 改造（僅改標題）

## 動機 / Motivation

GensparkLike 最初是複製 Genspark.ai 視覺作為技能交換平台。經過市場回饋，決定將產品定位轉向 PM 文件自動化 — 這是一群 PM 的實際痛點，市場需求更明確。

## 成功標準 / Success Criteria

1. 首頁呈現 PM EVO Ecosystem 品牌形象，版面無斷版
2. 主題切換器正常運作（16 色主題）
3. workspace 連結正確指向 `workspace/`
4. 所有 openspec 文件符合專案規則（R3/R4/R5/R7/R12/R13）
