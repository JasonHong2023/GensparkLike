# 複製 Genspark UI / Copy Genspark UI

> 狀態 / Status: Pending
> 建立 / Created: 2026-06-19

---

## 摘要 / Summary

以 Genspark.ai 的視覺設計（排版、色彩、字體、元件風格）為基礎，
建立 GensparkLike 技能交換平台（Skill Exchange Platform）的首頁與頁面
Build the GensparkLike Skill Exchange Platform homepage with Genspark.ai's visual design system.

## 範圍 / Scope

### 包含 / Included

1. 首頁 Landing Page（完整複製 Genspark 版面風格）
2. 品牌設計系統（色彩、字體、元件）
3. 靜態 HTML + CSS 實作（純前端，無框架）
4. RWD 響應式設計

### 不包含 / Excluded

1. 後端功能（登入、註冊、交易）
2. 真正的技能交換邏輯
3. 資料庫

## 動機 / Motivation

Genspark 的設計語言（簡潔、現代、futuristic 深色主題）非常適合用來呈現
技能交換平台的高品質形象，先複製視覺後置換內容。

## 設計參考 / Design Reference

- https://www.genspark.ai/
- https://www.genspark.ai/brand/genspark

## 成功標準 / Success Criteria

1. 首頁在視覺上與 Genspark 風格一致
2. 內容已置換為技能交換平台資訊
3. 在瀏覽器打開時版面正確、無斷版
